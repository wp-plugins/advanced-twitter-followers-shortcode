== Description ==

Thanks for installing our developed wordpress plugin.

Advanced Twitter Followers Shortcode is as much like facebook like box. As Facebook Like box show the thumbnails of peoples who do like the facebook page. As like much Advanced Twitter Followers Shortcode will show the followers thumbnail on your wordpress website.

This Plugin is tested on most of wordpress version and error checking make sure there's have no error found during using the plugin.

Note: Using this shortcode is very easy but you must have to create api for twitter to display the twitter followers on your wordpress website. Don't worry we have very nice documentation for everything you need to up the plugin on your wordpress website.

Features of Advanced Twitter Followers Shortcode

	* Absolutely Free.
	* Show twitter followers anywhere on your wordpress website using Advanced Twitter Followers Shortcode
	* Highly Customizable - as well can easily make tons of changes from backend.
	* Work with most of recent wordpress released versions.
	* Responsive - it make sure you able to enjoy our widget on mobile devices as well.
	* Can easily customize stylesheet.
	* Can add CSS class suffix too.
